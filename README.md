**kojiDld** is a high-performance desktop tool focused on extracting videos and audio from platforms like YouTube, TikTok, Instagram, and more. Designed to be fast and secure, it delivers files in the best possible resolution, clean (watermark-free), and features batch processing capabilities.

*(![Animação](https://github.com/user-attachments/assets/9c79e591-a835-4c42-90f0-7ce76c8fa22e)
)*

## Key Features

- **Clean & High-Quality Extraction:** Download videos in 1080p, 720p, or eco-friendly formats, always with the best available codec and without watermarks (depending on the source platform).
- **Native Audio Conversion:** Exclusive mode to extract only the audio and convert it to studio-quality MP3 (192kbps).
- **Batch Download Mode:** Paste up to 10 links at once and let the software process the entire queue automatically.
- **Forensic Security Scanner:** Before downloading, verify the integrity of the link. The app resolves the server's public IP, identifies the architecture (e.g., Nginx, Apache), and checks SSL/TLS encryption.
- **Real-Time Dashboard:** Track percentage, download speed (MB/s), and ETA with a clean, Dark Mode interface.

## How to Use (Easy Installation)

You do not need to install Python or know how to code to use kojiDld.

1. Go to the [Releases](https://github.com/xdevPablo/kojiDld/releases) tab.
2. Download the latest version (`kojiDld.exe`).
3. Double-click and start downloading!

## Tech Stack

This project was built using the following technologies:
- **[Python](https://www.python.org/)**: Base language.
- **[Flet](https://flet.dev/)**: Flutter-based UI framework, ensuring a smooth and modern interface.
- **[yt-dlp](https://github.com/yt-dlp/yt-dlp)**: The most powerful and up-to-date media extraction engine in the community.
- **[FFmpeg](https://ffmpeg.org/)**: Integrated multimedia processing tool to merge audio/video and generate MP3s.
